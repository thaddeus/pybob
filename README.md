pybob
=====

Python 3 interface for boblightd (LED controller)

Documentation
-------------
It's as simple as typing 
```python
import pybob
b = pybob.Boblights()
b.open()
b.set_light(b.get_lights()[0], "FF0000")
b.sync()
time.sleep(10)
b.close();
```

but you if you are interested, check out the module with executing `pydoc pybob`.
